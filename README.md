A responsive personal website built with HTML, CSS, and JavaScript, designed to showcase my skills, projects, and contact information. The website features a clean and modern design with interactive elements and a dark/light mode toggle.

Features

Dark/Light Mode: Users can switch between dark and light themes seamlessly.

Responsive Design: Works well on desktop, tablet, and mobile devices.

Navigation: Sticky header with smooth scroll links to different sections.

Skills Showcase: Visual badges highlighting technologies like HTML, CSS, JavaScript, React, Node.js, and Git.

Contact Form: Functional form with client-side validation and submission alert.

Interactive UI: Hover effects and smooth transitions for buttons and theme changes.

Technologies Used:

HTML: Provides the structure and semantic layout of the website.

CSS: Styles the website, handles responsive design, and implements dark/light mode using CSS variables.

JavaScript: Adds interactivity, theme toggle functionality, and form validation.
